Ketai Library


Collaborative Workflow and Documentation for Ketai Library


Ketai Library : Create moblie apps with Processing. http://ketai.org

Workflow built with Nodejs. Now live on Github

  • Github based workflow keeps the library reference updated. It converts Javadocs in Ketai Source to markdown files.
  • The website content is also managed as markdown files.
  • Upon any changes in Javadocs or website files, the server automatically renders fresh static HTML pages.
  • The HTML content is automatically synced with another Github Repo hosted on ketai.org

Libraries used:

  • Node.js
  • Chokidar
  • Javadocs to Markdown
  • Woods

Contributed to the Rapid Android Development book, Authored by Daniel Sauter
