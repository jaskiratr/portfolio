Vocabulary analysis of Top 100 books on Project Gutenberg:

About

The visualization below summarizes the vocabulary of the books publically published on Project Gutenberg. The data was gathered from the Project Gutenberg website by using Python. It was then analyzed using NLTK library for Python. The relevant data was baked into a JSON format. Some of the missing dataset fields were fetched from Goodreads API. Minor formating and cleaning up was done in sublime text and finally, the visualization was created using D3js.
